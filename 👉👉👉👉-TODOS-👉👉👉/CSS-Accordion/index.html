Smooth CSS-Only Accordions

What is CSS Grid animation?

CSS Grid animation uses the grid-template-rows property to smoothly transition between 0fr (collapsed) and 1fr (expanded) states. This creates natural height animations without JavaScript.

How does it work?

The technique works by:

  • Setting grid-template-rows: 0fr for closed state
  • Changing to grid-template-rows: 1fr when opened
  • Using CSS transitions to animate between states
  • Overflow hidden prevents content from showing during animation
Why use this method?

This approach is better than max-height hacks because:

  • No need to guess maximum content height
  • Animations are smooth regardless of content size
  • More predictable and reliable behavior
  • Works with dynamic content
Browser support

This technique has excellent browser support. The details/summary elements work in all modern browsers, and CSS Grid animations are supported in recent versions. Older browsers will still get functional accordions, just without smooth animations.
